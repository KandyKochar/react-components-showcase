# React Components Showcase

A collection of reusable React components built with TypeScript demonstrating core React patterns including state management, props, API integration, and list rendering.

## Overview
Four standalone TypeScript React components, each showcasing a different fundamental React concept. Compatible with Next.js App Router using the 'use client' directive.

## Components

### Counter
Interactive counter with increment, decrement, and reset functionality using useState.

### TodoList
Dynamic todo list with add, toggle completion, and delete operations. Demonstrates list state management and conditional rendering.

### UserProfile
Typed profile card component with required and optional props using TypeScript interfaces.

### ApiIntegration
Data-fetching component that loads posts from a public REST API, demonstrating useEffect, loading states, and error handling.

## Technologies Used
- **Framework:** React 18 / Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## Usage

Copy any component into your Next.js project's components folder:

```tsx
import Counter from './components/Counter'
import TodoList from './components/TodoList'
import UserProfile from './components/UserProfile'
import ApiIntegration from './components/ApiIntegration'
```
